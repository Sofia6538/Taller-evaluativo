¿Qué es Cloud y sus Modelos de Entrega?

Definición de Cloud

Cloud Computing es un modelo de prestación de servicios informáticos a través de internet, donde los recursos como servidores, almacenamiento, bases de datos y software se ofrecen bajo demanda y se gestionan de manera remota.

Modelos de Entrega Cloud

1. Software como Servicio (SaaS)

Permite a los usuarios acceder a aplicaciones alojadas en la nube a través de internet, sin necesidad de instalar nada localmente.

Ejemplo: Google Drive permite almacenar y editar documentos en línea.

2. Plataforma como Servicio (PaaS)

Proporciona un entorno y herramientas para que los desarrolladores creen, prueben y desplieguen aplicaciones sin preocuparse por la infraestructura subyacente.

Ejemplo: Heroku permite desplegar aplicaciones web fácilmente.

3. Infraestructura como Servicio (IaaS)

Ofrece recursos informáticos básicos como servidores virtuales, almacenamiento y redes, que los usuarios pueden gestionar y configurar según sus necesidades.

Ejemplo: Amazon Web Services (AWS EC2) permite crear y administrar servidores virtuales.
